# Student-Management-System
> The Student Management System is a desktop application developed using Python, Tkinter, and SQLite. It provides an easy and efficient way to manage student records through a user-friendly graphical interface. The application includes a secure login system and allows users to perform all basic CRUD (Create, Read, Update, Delete) operations on student data.
> This project was developed to demonstrate GUI development, database integration, and Python programming concepts. It is suitable for educational institutions or as a beginner-friendly database management application.
